Discription

This project presents a calculator with a gui that allows you to perform operations with real numbers, plot functions, and use a loan calculator.

Main part

There is a few binary, unary operators and functions. The manual only gives an example of some of the functions and operators that are not intuitive.

  1. % - Binary operator that calculates the remainder of a division
  2. ^ - Binary operator that calculates the power of number
  3. ln - Natural logarithm function
  4. lg - Base 10 logarithm function
  5. x - The value that will be substituted in the calculation process, or the graph of the function will be built on it
  6. pi - constant, that equal 3.14
  7. e - constant, that equal 2.72
  8. AC - all clean
  9. <- - delete one symbol

    10. The expression after the function is followed by parentheses on both sides

X value

When you input x to input line, there is opens new window, where you can input the value of x. Or you can set the scope and range of the function and see the resulting graph

Credit calculator

To use credit calculator change mode on menu bar. Then input all data about credit and get result.

If some parametrs of input anywere were incorrect, you will get the message of synatx error.
